# Fun with Gemini

This project test Gemini model capabilitis through various types of prompt and test varios types of capabilitis.

# How to run
1. Clone the repository
2. Install the depedencies using `npm install`
3. Create a `.env` file in the root directory and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
4. Run the project using `npm start`
5. Follow the instructions in the console to test various capabilities of the Gemini model.
